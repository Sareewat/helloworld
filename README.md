# Moon Bay ASCII Art

A simple Python project that prints a cinematic ASCII artwork of a ship sailing under the moonlight.

This project is created as a beginner-friendly Python practice project.
It uses basic `print()` statements and a raw multi-line string to display ASCII art in the terminal.

## Features

* Displays a large ASCII art ship in the terminal
* Uses Python raw string syntax to preserve backslashes
* Beginner-friendly and easy to customize
* Good for practicing Git, GitHub, and Python basics

---

## How to Run

Make sure Python is installed on your computer.

Then run this command in the terminal:

```bash
python main.py
```

or

```bash
python3 main.py
```

## Example Code

```python
print('Hello World')

text = r"""
YOUR ASCII ART HERE
"""

print(text)
```

The `r""" """` syntax is important because the artwork contains many backslashes `\`.
Using a raw string helps Python display the artwork correctly.
